# Google Integrations

Google is one optional provider for location resolution.

## Geocoding

Purpose:

- Convert event address text to trusted venue coordinates when configured

Requires:

- `GOOGLE_MAPS_API_KEY`
- Geocoding API enabled in Google Cloud project

## Not Used

- Google is not the live runtime event source
- User auth collection is not stored in Google as primary app behavior
- Homepage event loading does not require Google geocoding
